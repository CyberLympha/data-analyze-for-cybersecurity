# Задачи по датасету "Титаник":

1. Проанализировать связь между признаками и визуализировать их в виде тепловой карты

2. Задача классификации: выжил/не выжил. Использовать две модели: логистическую регрессию и деревья решений.

2.1 Для каждой модели вывести метрики качества, изобразить матрицу "confusion".

2.2 Для модели деревьев решений построить кривую обучения - зависимость accuracy и precision от глубины дерева. Определить нужную глубину дерева и когда происходит переобучение.

2.3 Вы бы выжили на Титанике? Для каждой модели.

3. Задача регрессии.

3.1 Сделать предсказание стоимости билета.
3.2 Сделать предскзание возраста пассажира.

Для кадой задачи вывести все метрики качества (MAE, MAPE, ...)
